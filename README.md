# BBC News Text Classification using Naive Bayes

A Machine Learning project that predicts the category of BBC news articles using Natural Language Processing (NLP) and the Naive Bayes algorithm.

## Categories
- Business
- Sports
- Politics
- Technology
- Entertainment

## Features
- Text Cleaning
- Stopword Removal
- CountVectorizer
- Naive Bayes Classification
- Streamlit Web App
- Real-time Prediction

## Technologies Used
- Python
- Pandas
- Scikit-learn
- NLTK
- Streamlit

## Dataset
https://raw.githubusercontent.com/ronaldleonardo/Data_Transformer_Datasets/refs/heads/main/bbc_text_cls.csv

## Machine Learning Algorithm
- Multinomial Naive Bayes
